# Poc-pdf-textract

Building an OCR solution for document analysis with AWS Textract and AWS StepFunctions.

## Installation

Use the package manager npm to install the dependency and then run serverless to deploy in your AWS account.

```bash
npm install 
serverless deploy 
```
